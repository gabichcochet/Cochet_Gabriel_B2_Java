# Projet Java Bachelor 2 - Examen

Ce projet respecte les attentes de la consigne officielle du devoir Java avancé. La structure Maven est conservée avec les modules : `partie1`, `partie2`, `partie3` et `partie4`.

## Lancer le projet

1. Ouvrir le projet à la racine `Cochet_Gabriel_B2_Java`.
2. Exécuter la commande suivante :

	```bash
	mvn test
	```

3. Vérifier que tous les tests passent pour les 4 modules.

## Tests

- `mvn test` doit être utilisé pour valider le projet.
